# Academic Manager Legacy - Com Problemas Plantados

Sistema legado em Python com dívida técnica proposital para uso em aulas de
Manutenção e Evolução de Software.

## Objetivo didático
Este projeto foi preparado para que os alunos possam:
- mapear fluxo de execução;
- identificar problemas funcionais e estruturais;
- registrar demandas de manutenção;
- analisar manutenibilidade com ferramentas automatizadas.

## Problemas plantados
- uso de `next()` sem tratamento;
- função com múltiplas responsabilidades;
- duplicação de cálculo;
- constantes hardcoded;
- `except:` genérico;
- classes de domínio pouco integradas ao fluxo;
- inconsistência intencional no `dados.json`.

## Execução
```bash
python app.py
```
